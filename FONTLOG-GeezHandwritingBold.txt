FONTLOG

Geʾez Handwriting bold font family
==================================
This file provides detailed information on the Geʾez Handwriting bold font.
This information should be distributed along with the Geʾez Handwriting
bold font and any derivative works.

Basic Font Information
----------------------
This "Geʾez Handwriting" bold typeface is derived directly fromthe
"Geez Handwriting" bold typeface designed by Educational Fontware, Inc.
(https://www.educationalfontware.com/)

The Geʾez Handwriting bold weight typeface of the basic style can be
used for emphasis is also part of the collection and uses the same
font name, Geʾez Handwriting, as the regular style.

The font is part of a collection of 14 typefaces developed to help teach
handwriting to children. The other fonts of the collection will have
separate FONTLOG files that track changes to those fonts specifically.
The complete tyepface family comprises:

     * Ge'ez Handwriting Regular
     * Ge'ez Handwriting Bold (this font)
     * Ge'ez Handwriting with Rules
     * Ge'ez Handwriting with Arrows
     * Ge'ez Handwriting Arrows Rules
     * Ge'ez Handwriting Arrows Only
     * Ge'ez Handwriting Dots
     * Ge'ez Handwriting Dots Rules
     * Ge'ez Handwriting Dots Arrows
     * Ge'ez Handwriting Dots Arr Rule
     * Ge'ez Handwriting Outlines
     * Ge'ez Handwriting Outlines Rule
     * Ge'ez Handwriting Out Arrows
     * Ge'ez Handwriting Out Arr Rules
