FONTLOG

Geʾez Handwriting Dots Arr Rule font family
===========================================
This file provides detailed information on the Geʾez Handwriting Dots
Arr Rule font. This information should be distributed along with the Geʾez
Handwriting Dots Arr Rule font and any derivative works.

Basic Font Information
----------------------
This "Geʾez Handwriting Dots Arr Rule" regular typeface is derived
directly from the "Geez Handwriting Dots Arr Rule" typeface designed
by Educational Fontware, Inc. (https://www.educationalfontware.com/)

The Geʾez Handwriting Dots Arr Rule regular typeface applies directional
arrows to the to the dots style to indicate stroke direction and adds
writing guides ("rules") as would be found in a notebook.

The font is part of a collection of 14 typefaces developed to help teach
handwriting to children. The other fonts of the collection will have
separate FONTLOG files that track changes to those fonts specifically.
The complete tyepface family comprises:

     * Ge'ez Handwriting Regular
     * Ge'ez Handwriting Bold
     * Ge'ez Handwriting with Rules
     * Ge'ez Handwriting with Arrows
     * Ge'ez Handwriting Arrows Rules
     * Ge'ez Handwriting Arrows Only
     * Ge'ez Handwriting Dots
     * Ge'ez Handwriting Dots Rules
     * Ge'ez Handwriting Dots Arrows
     * Ge'ez Handwriting Dots Arr Rule (this font)
     * Ge'ez Handwriting Outlines
     * Ge'ez Handwriting Outlines Rule
     * Ge'ez Handwriting Out Arrows
     * Ge'ez Handwriting Out Arr Rules
